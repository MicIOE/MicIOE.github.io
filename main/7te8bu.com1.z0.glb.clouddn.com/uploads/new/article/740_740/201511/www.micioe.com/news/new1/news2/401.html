中国版特斯拉？游侠汽车成色几何

“游侠”，第一次听到这个名字时，你可能与笔者有一样的感觉，这不是一款国产游戏吗？

“游戏”成真，50 个人从零开始，经过 482 天的疯狂追逐，造出了他们梦想中的电动汽车。明年开始预定，后年就可提车，“游侠 X ”发布会现场掀起的阵阵躁动，让黄修源迎来职业生涯中的最高潮。这一切，只因这位年纪比“天才少年”李一男还小得多的 85 后连续创业者造出了首个“互联网汽车品牌”—— 游侠。

山寨特斯拉，黄修源的游侠 X 干得漂亮吗？



同为“天才少年”，在做游侠汽车前，黄修源走的是一条不同于李一男的创业“小道”。


2008 年，黄修源从中国传媒大学传播学专业毕业后，经过豆瓣短暂实习，随后创办一家鞋类 B2C 公司“番茄树”；一年以后，团队转向导购，创办了可能是国内最早的商品导购网站“翻东西”。2012 年，黄修源团队再变方向，开始接手运营百度旗下的搞笑视频以及手机早晚报等业务。一年之后，旋即做起投资人的黄修源，投资并参与了一家以移动话费充值为业务核心的公司，并实现了超千万的营收。


翻看黄修源的过往履历，一个字总结，就是在不停地“变变变”。相对来看，他之前做的 N 多项目中没有一个算得上突出，短则几个月长则一年天气，就去“跟风”做下一个新的业务了。甚至，他的这些过往项目与汽车业都没半点交集，但这丝毫不妨碍他在“闭关” 482 天后拿出一款“山寨版”特斯拉原型来扬名天下。

7 月 26 日，在一帮大众媒体的见证下，舞台中心的游侠汽车创始人黄修源操着苹果范儿十足的 Keynote “隆重”推介了旗下第一款电动车游侠 X 的原型车。

此前接受媒体采访时，黄修源表示：“我们就是在模仿特斯拉！”那么这款“致敬之作”又成色几何呢？

性能一般，主要参数与特斯拉现有最低端车型基本持平。

2017 年上市的游侠 X 续航里程可达 460km，百公里加速需 5.6s。作为对比，特斯拉刚刚更新的 Model S 产品线中，最低端的 70 车型（国内售价为 64.11 万）的续航里程为 420km，百公里加速为 5.8s。两年后才能买得到的游侠 X 在这项重要性能上，才刚刚超过特斯拉现有最低端电动车的参数。游侠汽车的超跑进击之路，离颠覆是不是还差着 N 多特斯拉车型呢？即便游侠汽车的“终极大招”价格可能将远低于特斯拉，但现阶段想买电动汽车的玩家，没多少人会过分看重价格因素。此外，全新的特斯拉 Model 3 系列也将于明年初登场，其最低售价可能下探到 3.5 万美金左右。

内外部设计洋溢浓浓特斯拉味，更有华而不实的噱头来添彩。

外部来看，游侠 X 整车采用流线型设计，四门轿跑，外观几多特斯拉身影。有媒体报道称，游侠 X 的底盘基于特斯拉 Model S 逆向工程而来。更加“有趣”的是，在汽车前部的内循环进风口处，中国的互联网年轻人给游侠 X 装了一张基于 LED 显示屏的可以表达“情绪”的前脸。玩家可以根据自己的喜好，自行设计汽车“前脸”显示的图案。如果被不明真相的群众看到了汽车可爱的笑脸，会不会以为哪家又在办婚礼呢......

再看内部，游侠 X 配备了与特斯拉相同的 10.2 英寸液晶仪表盘和 17.3 英寸 1080P 高清中控显示屏。除了科技感，为了把内部装饰的格调也提上去，游侠 X 的座椅与操控台分别采用了与宾利同款的原版 Nappa 皮革以及轿跑专用的 Alcantara 面料。

特斯拉有的科技元素，游侠原型车给你了；特斯拉没有的奢华，游侠也给你了：看了黄修源“倾情”呈现给大伙的游侠 X，你是不是又产生了“物美价廉”的幻想？

汽车装了个 Android 系统，就变智能了？

汽车不智能，还闹啥革命。游侠 X 采用 KITT OS，这是一款基于 Android 5.1 专为汽车定制的交互系统。整车控制，玩家只需与中控大屏进行交互。KITT OS 拥有智能语音系统，可以自主学习用户的语言习惯。KITT OS 还能让玩家通过智能手机、手表等移动设备与游侠 X 进行信息交流，亦可远程控制或手表开锁。预装行车记录仪、ADAS 高级驾驶系统以及 MobileEye 辅助驾驶系统，游侠 X 给出的“智能化套餐”看起来颇为丰盛。

客观来说，对于一辆售价并不高昂的电动车来说，能实现这些粗浅的智能化功能，而且还打包了一干新东西，已经算很不错了。但对于想要颠覆电动车界甚至叫板特斯拉的游侠 X 而言，这些所谓的智能化新功能，不仅特斯拉早就有了，就连搭载 Android Auto 抑或 CarPlay 的车型也能基本实现。与其说 KITT OS 是一个新系统，倒不如说它更像是 Android Auto 的汉化版。Android Auto 刚刚起步，KITT OS 又能牛逼到哪里去呢？

“里程焦虑”这个未来问题，黄修源没怎么说。

制约特斯拉更大发展的“里程焦虑”问题，本来应该是“友商”们在开发布会时着力进行攻击的地方，但黄修源在发布会上好像压根就不想提这个事儿。对于购买电动车极为重要的充电问题，黄修源也是轻描淡写地就过去了：游侠电动同时支持家用充电和超级快充，最快充电可在半小时内完成 270km 里程准备，未来用户在购买汽车同时可享受家用充电安装等增值服务。

换个角度来看，对于还停留在原型车展示阶段的游侠汽车而言，当务之急是尽快拿到大额融资，然后再联合各方力量把游侠 X 做出成品，并早日争取走上量产之路。如果先前“三天两头”就写创业企划书的黄修源，能走完这么多步，实现人生最长的一段创业旅程，或许也就够了。至于那些“焦不焦虑”的续航里程，还是等游侠电动汽车真能走到量产下线那一刻，再去抒发新的情怀吧。

互联网味儿十足，游侠汽车能开上道吗？

游侠 X，这款“互联网汽车”赶在李想以及一干大厂的汽车改造大业前公开亮相了。因为是国内“互联网汽车”的首秀，而且还有一位传播学专业背景的 CEO 来掌舵，所以笔者的朋友圈（主要是科技记者、编辑）在发布会同期被疯狂刷屏。这场花了 482 天精心准备的发布会，不仅超乎寻常地完成了传播上的使命，而且也让笔者第一次知晓了游侠的大名以及黄修源背后的故事。但游侠汽车以及所谓的“互联网汽车”概念，对于笔者而言依然像是个新兴的事物。新兴并不可怕，可怕的是看不懂的背后，以及其与旧现象的颇多共同之处。

482 天真能倒腾出一辆全新的电动汽车？

对于游侠汽车而言，或许这个问题本身就是一个伪命题。因为游侠 X 原型车的底盘可能是基于特斯拉 Model S 逆向工程而来的。在知乎上更有人声称，发布会上高调亮相的那辆游侠 X 原型车就是基于特斯拉 Model S 改装而来的，”改了前脸和尾灯、把车标换成自己的……“不过，黄修源随后否认了外界对于游侠 X 原型车底盘直接采用特斯拉的质疑，“底盘我们还真的没有用特斯拉原版的”。

不管游侠 X 原型车是不是直接改装特斯拉 Model S 而成，至少有一点可以明确：游侠 X 原型车在很大程度上抄袭了特斯拉电动车颇为成熟的设计“原型”。站在巨人肩上，你就能比牛顿更伟大？发布会上，黄修源这位热血青年扬言要做一辆价格亲民并且好用的电动汽车，但你发布的原型车上对“山寨”行径的不加掩饰，就是要做“好车”的态度吗？

此外，得益于马斯克颇为大方的特斯拉“专利开源”运动，特斯拉电动车和电池的知识产权可以被世界上任何一家公司，甚至任何一个人拿来免费使用。游侠 X 的电动车以及电池技术中究竟有多少是直接“拿来主义”的呢？

虽然 482 天就倒腾出了一辆游侠 X 原型车，但请黄修源告诉大家，究竟哪些是你凭借互联网思维创造出来的新东西？国内首款“互联网汽车”究竟有哪些革新的地方？

除过抢占舆论高地的“互联网汽车”名头，游侠汽车还能亮出多少真家伙？

汽车本是一个高门槛行业，但借助当下大行其道的互联网思维，一波互联网大公司都想挤进来玩一玩造车游戏，万一实现呢？但对于黄修源这样的草根创业者，没有行业积淀，单凭兴趣使然，几无胜算可能。黄修源可不这么觉得，还把“世界所有的美好来源于被嘲笑的梦想”当成了自个的 Slogan 来大加宣扬。

这位在汽车业可能算是最年轻的 CEO ，要做的“不仅仅是一台交通工具，而是智能、好玩、颠覆的互联网跨界科技潮物”。甚至在他看来，特斯拉在交互上并不完美，而游侠 X 做了很多创新。就笔者目前掌握的资料来看，游侠 X 身上所谓的创新多为华而不实的噱头，倒是首款“互联网汽车”的名头成了国内汽车理论界的一大“创新”。

然而，披上“互联网”的华丽外衣后，汽车依然还是一辆汽车。传统汽车在设计以及量产上所面临的一系列问题，游侠 X 依然也要面对。对于一个现在依然还不到百人的互联网运作小团队来说，造车大业需要的相关软硬件条件以及一帮专业人才，是其不可承受之重。此外，汽车公司创始人必要的资历经验和从业背景，黄修源一个都不沾。核心管理团队中，也没见那位汽车业大咖为其站台。

除过凭借一台像极了特斯拉的概念车抢占了舆论的高地，游侠还能亮出多少真家伙？

iPhone 6 能被华强北山寨，但特斯拉真能被山寨吗？

有评论认为，李想等创业系大佬、互联网大公司以及传统汽车厂家，均未拿出一款公开亮相的可以“比肩”特斯拉的国产电动汽车，而游侠这家名不见经传的小公司却率先拿出了一款亮相颇为“惊艳”的游侠 X，这已实属难得。对于国内的互联网小公司做大汽车，我们需要给予这帮年轻人相当程度的宽容和理解。这种论调，看似有理，但实则是在鼓励山寨。iPhone 是全世界最好的手机，那我们也该对中国大地方兴未艾的翻新以及山寨  iPhone 持有同情和理解之心？殊不知，2000 多的 iPhone 6 Plus 害了多少只想用台好手机的不明真相群众！

再看手机发展史：功能机时代，是山寨机发展最为鼎盛的时期，因为门槛低；来到智能机时代，做手机的门槛突然变高了，再加上小米们掀起的“性价比”风暴，山寨机市场虽未死绝，却大不如从前。黄修源想把游侠电动汽车，做成放大版的“超跑手机”。可能他也认为，山寨机靠模仿诺基亚或摩托罗拉功能机的样子就能成就当时的辉煌，现在该轮到汽车业经历这一浪潮了；而且错过了初期的山寨好时机，未来就没机会了。但手机可以山寨，汽车真能山寨得了？姑且不说汽车上路的安全以及监管风险，汽车上数以万计的零部件按照模仿特斯拉来的设计与工艺进行组装就是一件大难事。传统汽车市场从来就没有出现过一家靠模仿国外车厂成功的案例，你游侠凭啥能做到模仿人家特斯拉？

过去传统车厂做不到模仿国外车厂，黄修源团队现在却想通过“互联网思维”做出的“互联网汽车”来实现模仿特斯拉的“壮举”。但就目前掌握的信息，笔者完全看不透黄修源团队打算怎样做出一款汽车成品出来，要知道这可是第一步呀。要想做出一个山寨原型车，一个发烧级玩家通过改装现有车型就能实现近似相同的展示效果。但要想做出一辆可以开上道的普通汽车就真的堪比登天了，尤其是怀着模仿特斯拉的初衷去做，可能最后出现的是一个不伦不类的“怪胎”产品。

我们设想黄修源团队经过艰难险阻（极有可能是找代工厂装配生产）做出了一辆普通汽车，那么游侠 X 真正的挑战就来了：如何实现智能化新功能与汽车传统元素的有机结合，让智能化体验真正提升车主的驾乘体验；硬件的安全性以及系统的稳定性如何不因智能化功能的加入而大打折扣。特斯拉电动车内在的这些真功力，你游侠也能“照猫画虎”过来吗？

面对游侠 X 亮相后引来的非议，黄修源也在知乎上晒出了自个的回应：

我们在第一期的选择并不是要求所有的事情都做到 100% 的没有问题。我们第一期想做好的事情是，设计，车机操作系统，电池包。

设计、操作系统、电池包，这是黄修源给出的“互联网汽车”游侠 X 着力攻克的三个方面。设计见仁见智；操作系统有了 Android Auto 在先，可能也不算是什么难事；至于电池包，有了特斯拉开源专利的鼎力支持，或许也能轻松搞定。所以不难看出，游侠 X 原型车并没有黄修源在发布会上渲染的那般“颠覆”。发布会后，他也在知乎上直言，“大家没有必要要求一个成立 15 个月的团队能超过奔驰宝马，或者任何一家 10 年以上的汽车公司，这不实际”。
